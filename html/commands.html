Ayko Suporte Comandos Extras

Meus comandos

Obs: Para utilizá-los, você precisa sempre adicionar o meu prefixo "!", senão algo muito ruim pode acontecer... ≽^•⩊•^≼

!date

Utilizado para informar a data e hora atual.

!date

!avatar

Utilizado para mostrar sua foto/gif de perfil ou de outro usuario.

!avatar ou !avatar [@usuario]

!play

Utilizado para tocar musicas de acordo com o nome da mesma ou utilizando um link direto do youtube.

!play nomeDaMusica OU !play https://youtube.com/musica

!cat

Utilizado para receber uma imagem aleatória de um gatinho muito fofinho. :3

!cat

!setStream

Utilizado para receber notificações sempre que o streamer configurado entrar ao vivo.

!setStream

!revStream

Utilizado para remover as notificações de determinado streamer já configurado.

!revStream

!setWM

Utilizado para setar um canal de texto para mensagens de entrada de usuários.
Obs: Precisa da permissão de Gerenciar Servidores.

!setWM

!setGM

Utilizado para setar um canal de texto para mensagens de saída de usuários.
Obs: Precisa da permissão de Gerenciar Servidores.

!setGM

!setChannel

Utilizado para setar o canal que terá a configuração de cargo por reação.

!setChannel

!setMessage

Utilizado para setar uma mensagem específica para receber o cargo por reação.

!setMessage [mensagemID]

!setReact

Utilizado para definir cargos reagindo em mensagens.

!setReact

!revReact

Utilizado para retirar o cargo por reação configurado em uma mensagem.

!revReact [mensagemID] [emojiEscolhido]

!autoRole

Utilizado para adicionar um cargo especifico automaticamente aos usuários que ingressarem no servidor.

!autoRole

!revAutoRole

Utilizado para remover as configurações do comando autoRole.

!revAutoRole

!sendEmbedWelc

Utilizado para enviar o modelo do Embed de Boas-Vindas atualmente utilizado.

!sendEmbedWelc

!edTitleWelc

Utilizado para personalizar o título do Embed de Boas-Vindas.

!edTitleWelc [MensagemTítulo]

!edColWelc

Utilizado para personalizar a cor na lateral esquerda do Embed de Boas-Vindas.

!edColWelc [#fff]

!edImgWelc

Utilizado para personalizar a imagem central do Embed de Boas-Vindas.

!edImgWelc [ImagemURL]

!edDescWelc

Utilizado para personalizar a descrição do Embed de Boas-Vindas

!edDescWelc [MensagemDescricao]

!edFootWelc

Utilizado para personalizar o rodapé do Embed de Boas-Vindas.

!edFootWelc [MensagemRodapé]

!sendEmbedBye

Utilizado para enviar o modelo de Embed de Despedidas.

!sendEmbedBye

!edTitleBye

Utilizado para personalizar o título do Embed de Despedidas.

!edTitleBye [MensagemTítulo]

!edColBye

Utilizado para personalizar a cor lateral do Embed de Despedidas.

!edColBye [#fff]

!edImgBye

Utilizado para personalizar a imagem central do Embed de Despedidas.

!edImgBye [ImagemURL]

!edDescBye

Utilizado para personalizar a descriçao do Embed de Despedidas.

!edDescBye [MensagemDescricao]

!edFootBye

Utilizado para personalizar o rodapé do Embed de Despedidas.

!edFootBye [MensagemRodapé]

!crush

Utilizado para analisar a porcentagem de ambos usuários possuirem algo em comum.

!crush [@usuario1] [@usuario2]

!gif

Utilizado para receber um gif aleatório.

!gif

!kiss

Utilizado para enviar um beijo no usuário mencionado.

!kiss [@usuario]

!ping

Utilizado para retornar a latência entre o servidor e o cliente (você).

!ping
